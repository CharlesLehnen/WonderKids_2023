# Zooming into STEM: Designing Curriculum Around Guest Presentations in an Online After-School STEM Program for Title I Elementary Students

## Overview

This is a repository for the code underlying the plots and statistical analyses presented in the [Innovations in Science Teacher Education](https://innovations.theaste.org/index.php/iste/index) 2026 article [**"Zooming into STEM: Designing Curriculum Around Guest Presentations in an Online After-School STEM Program for Title I Elementary Students"**](https://innovations.theaste.org/index.php/iste/article/view/454?fbclid=PAVERFWASyQ45wZG9mAmV4dG4DYWVtAjEwAHNydGMGYXBwX2lkDzEyNDAyNDU3NDI4NzQxNAABp0RrR4z-wIByikBXaOLzwtcHDa6B2tFVbla57oPp8CvD50If9VHAX7xgd49B_aem_O_dJvxOlCygwzoSMWEU9aQ) 

## Repository Structure

### Folders

- **code/**: This directory contains all the code scripts used in the project.

  - [Wonderkids_Metrics_Fall_2023.ipynb](code/Wonderkids_Metrics_Fall_2023.ipynb) is an early iteration of the analysis that is antiquated
  
  - [WonderKids.Rmd](code/WonderKids.Rmd) presents the code that was used to produce the plots and analyses presented in the article

- **data/**: Houses raw, immutable data files used in the analyses. Student names are anonymized. 

- **documents/**: Contains a PDF of the article

- **output/**: Stores all the outputs generated by the scripts, such as processed data, figures, and tables, reflecting the end results of the data analysis pipeline.


## Getting Started

All final code (written in R) can be found in [WonderKids.Rmd](code/WonderKids.Rmd)
